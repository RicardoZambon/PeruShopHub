## Codebase Patterns
- All components are **standalone** (no NgModules) with `standalone: true`
- Use `CommonModule` import for `*ngFor`, `*ngIf` directives
- Barrel exports in `shared/components/index.ts` ‚Äî export both component and types
- SCSS uses `@use '../../../../styles/mixins' as m;` for responsive breakpoint mixins
- CSS custom properties from `tokens.css` for all design tokens (e.g., `var(--surface)`, `var(--neutral-200)`)
- IntersectionObserver for infinite scroll: run outside NgZone, use scroll container as root, re-enter zone only to emit events
- Angular build via `npx ng build --configuration=development` ‚Äî takes ~95 seconds
- Template file convention: separate `.html` file with `templateUrl`
- Style file convention: separate `.scss` file with `styleUrl` (singular, not `styleUrls`)
- **Service layer pattern**: All business logic in Application/Services, controllers are thin HTTP adapters
- **Exception pattern**: NotFoundException (404), AppValidationException (400 with errors dict), ConflictException (409) ‚Äî caught by GlobalExceptionFilter
- **Validation pattern**: Collect all errors into `Dictionary<string, List<string>>` before throwing AppValidationException
- **Optimistic locking**: Version property + ConcurrencyToken on Product, Category, PurchaseOrder, Supply ‚Äî set OriginalValue, increment, catch DbUpdateConcurrencyException
- **Role-based auth**: Admin for Settings, Admin/Manager for mutations, any authenticated for reads
- **Build command**: `dotnet build src/PeruShopHub.API/PeruShopHub.API.csproj /nodeReuse:false /p:UseSharedCompilation=false -o /tmp/perushophub-build` (VS Code locks default bin/)
- Application project needs explicit EF Core 9.* package reference due to version resolution issues
- BCrypt.Net-Next already referenced in Infrastructure project
- Product interface defined inline in `product.service.ts`, Category in `models/category.model.ts`, Supply in `supply.service.ts`, PO in `purchase-order.service.ts`
- Error interceptor uses functional `HttpInterceptorFn` pattern (not class-based)
- Remote is `RicardoZambon/PeruShopHub` on GitHub ‚Äî use for badge URLs and ghcr.io paths
