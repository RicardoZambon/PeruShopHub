## Plano de Execucao em 3 Fases

### Fase 1 - Teste (Semanas 1-3) | Gastar R$ 500-800

1. Escolher 5-8 produtos baseado nos mais vendidos do ML (filtro "mais vendidos")
2. Comprar 5-10 unidades de cada para testar
3. Criar anuncios otimizados (fotos fundo branco, titulo com palavras-chave)
4. Ativar ML Ads com R$ 10-20/dia nos melhores anuncios
5. Responder perguntas rapido (melhora posicionamento no algoritmo)

### Fase 2 - Validacao (Semanas 3-6)

1. Identificar os 2-3 produtos que mais venderam
2. Investir R$ 1.500-2.000 reforcando estoque dos vencedores
3. Eliminar produtos que nao venderam (liquidar com desconto se necessario)
4. Comecar a usar o Mercado Livre Full para os campeoes de venda

### Fase 3 - Escala (Mes 2-3)

1. Reinvestir 100% do faturamento nos produtos validados
2. Buscar fornecedores com melhor preco (migrar de AliExpress para 1688, ou de 25 de Marco para fabrica)
3. Adicionar 2-3 novos SKUs para testar novamente
4. Considerar abrir MEI (emissao de NF, acesso a Coleta, Mercado Lider)

## Ferramentas Uteis

- **Nubimetrics** (teste gratuito): Volume de buscas e vendas por categoria no ML
- **Google Trends Brasil**: Sazonalidade de produtos
- **TikTok / Instagram**: Identificar produtos virais
- **ML filtro "Mais vendidos"**: Analisar concorrencia e demanda

---

## Dicas Essenciais

1. **Fotos profissionais sao obrigatorias** - Fundo branco, multiplos angulos
2. **Titulo otimizado** - Use palavras-chave que o comprador pesquisa
3. **Precifique corretamente** - Considere TODOS os custos (comissao + frete + embalagem + imposto)
4. **Comece com atacado nacional** - Importacao da China leva 30-60 dias; compre no Bras/25 de Marco enquanto aguarda
5. **Reinvista os lucros** - Nos primeiros 3-6 meses, reinvista 100% para aumentar estoque
6. **Abra um MEI** - NF, menos impostos, acesso a recursos do ML
7. **Nao aposte tudo em 1 produto** - Diversifique em 3-4 nichos
8. **Use ML Full desde o inicio** - Selo azul = mais visibilidade e vendas
